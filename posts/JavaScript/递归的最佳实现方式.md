这里factorial = null删除了对函数的引用，递归调用时会报错

 1. 可靠的方式，但是该方式在严格模式下禁用，

    	function factorial(num){
        	if(num<=1){
            	return 1;
        	}else{
          		return num*arguments.callee(num-1);
        	}
    	}

 2.所以使用命名函数表达式来达成相同的结果：

    	var factorial = (function f(num){
        	if(num<=1){
            	return 1;
        	}else{
          		return num*f(num-1);
        	}
    	});
